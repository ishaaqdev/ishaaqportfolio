# Mohammed Ishaaq — Portfolio

Personal portfolio site built with [Astro](https://astro.build) and [Tailwind CSS](https://tailwindcss.com). Deployed at [ishaaqdev/ishaaqportfolio](https://github.com/ishaaqdev/ishaaqportfolio).

## Stack

- **Framework:** Astro v4 (static output)
- **Styling:** Tailwind CSS v3 with custom CSS variables for the design system
- **Fonts:** Poppins, Roboto (Google Fonts) + Material Symbols
- **Deployment:** GitHub Pages / static host

## Pages

| Route | Description |

| `/projects` | Full project listing with animated cards |

## Running locally

```bash
cd C:\ishaaq\projects\portfolio
npm install
npm run dev
```

Then open `http://localhost:4321` (or whichever port Astro assigns).

## Build

```bash
npm run build
```

Output goes to `dist/`.
